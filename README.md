# Сапевари — Сайт о грузинской кухне



---

### **Сайт ресторана «Саперави»**
Одностраничный сайт современного ресторана грузинской кухни, созданный с использованием чистого HTML/CSS а так же Frondend и backend. Каждая страница демонстрирует различные аспекты веб-разработки и пользовательского опыта.

### **Список страниц и функций**
**Главная страница (index.html)** — Визитная карточка ресторана с хитами кухни, фото-галереей и призывом к действию.

**Меню (menu.html)** — Интерактивный каталог блюд с динамической загрузкой данных из JSON, фильтрацией и возможностью добавления в корзину.

**О нас (about.html)** — Страница с историей ресторана, философией и галереей интерьера.

### **ДЕТАЛЬНОЕ ОПИСАНИЕ ИНТЕРАКТИВНОГО МЕНЮ**
**Ключевые возможности**

**Для гостей:**
- 3 категории блюд с фильтрацией:
  - Закуски и салаты
  - Основные блюда
  - Десерты и напитки
- Динамическое отображение:
  - Загрузка данных из menu.json
  - Адаптивный дизайн для всех устройств

**Для администраторов (через консоль):**
- Простое управление меню через редактирование menu.json
- Быстрое обновление цен и описаний
- Легкая замена изображений в папке img/

**Технические требования и запуск**
Следуйте этим шагам, чтобы запустить сайт на вашем компьютере.

**Минимальные требования:**
- Любой современный браузер (Chrome 90+, Firefox 88+, Safari 14+)
- Локальный сервер (например, Live Server для VS Code)

**Пошаговая инструкция**
1. **Скачайте файлы проекта:**
   - Скопируйте все файлы из структуры ниже в одну папку

2. **Установите библиотеки:**
   - Откройте проект в VS Code
   - Установите библиотеки:npm i

3. **Запустите локальный сервер:**
   - Запустите терминал
   - Введите команду: npm run dev"

4. **Откройте браузер:** Сайт откроется по адресу `http://localhost:5500` (или другому порту).

### **Структура проекта**
```
saperavi-restaurant/
├── data/                    # Данные сайта
│   └── menu.json           # Данные меню в формате JSON
├── img/                    # Изображения сайта
│   ├── hero-bg.jpg         # Фон главной страницы
│   ├── dish-1.jpg          # Фото блюд
│   ├── dish-2.jpg
│   ├── interior-1.jpg      # Фото интерьера
│   └── logo.png            # Логотип ресторана
├── node_modules/           # Зависимости
├── about.html              # Страница "О нас"
├── index.html              # Главная страница
├── menu.html               # Страница меню
├── menu.json               # Основной файл данных меню (дублирование)
├── package-lock.json       # Файл блокировки версий npm
├── package.json            # Конфигурация npm-проекта
├── script.js               # Основная логика сайта (JavaScript)
├── server.js               # Простой сервер на Node.js (опционально)
└── style.css               # Стили сайта (CSS)
```

### **Описание файловой структуры**
**HTML-файлы (основные страницы)**
- **index.html** — Главная страница с навигацией, герой-секцией, хитами кухни
- **menu.html** — Интерактивное меню с фильтрацией и корзиной
- **about.html** — Информация о ресторане, команде, контакты

**Данные (data/ и JSON-файлы)**
- **menu.json** — Структурированные данные всех блюд (название, описание, цена, категория)
- **data/** — Дополнительные данные для расширения функционала

**Медиа-ресурсы (img/)**
- **Фотографии блюд** — Высококачественные изображения всех позиций меню
- **Интерьер** — Фото залов ресторана
- **Логотипы и иконки** — Фирменная айдентика

**JavaScript файлы**
- **script.js** — Основная логика:
  - Загрузка и отображение меню
  - Работа с корзиной
  - Валидация форм
  - Анимации и интерактив
- **server.js** — Простой Node.js сервер для локальной разработки

**Стили (CSS)**
- **style.css** — Все стили сайта:
  - Адаптивная верстка
  - Анимации и переходы
  - Стилизация форм и кнопок

### **Особенности архитектуры**
**Модульность** — Четкое разделение HTML (структура), CSS (оформление), JS (логика)

**Производительность** — Минимальное количество HTTP-запросов, оптимизированные изображения
